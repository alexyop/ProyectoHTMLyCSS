Accesorios Discos Duros

Startech Carcasa SSD 2.5

Ahora puede tener acceso a sus datos en tiempo récord, gracias al rendimiento superior que le ofrece el estándar USB 3.1 Gen 2 (10 Gbps). Esta caja para una unidad de disco le hace posible disponer de una solución poderosa de almacenamiento de datos externa.

Startech S352BU313R Caja Externa RAID

Esta caja de unidad de disco Startech S352BU313R de doble bahía USB 3.1 facilita agregar almacenamiento externo de datos, que ofrece gran capacidad y alto rendimiento a tu ordenador portátil o de sobremesa. Esta caja RAID de doble bahía es compatible con dos unidades de disco SATA de 3,5", para obtener velocidades de transferencia de datos ultrarrápidas y modos RAID configurables.

Startech Base Dock

La estación de conexión para doble disco duro SATA HDD/SSD de 2,5"/3,5", modelo SDOCK2U33RE, permite duplicar/clonar una unidad de disco duro existente sin un sistema de ordenador anfitrión, o bien conectar e intercambiar, de forma simultánea, unidades de disco duro de un ordenador de sobremesa o portátil, a través de un único puerto eSATA o USB.
